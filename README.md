# Desafio de Lógica com JavaScript - DIO 🚀

Este repositório contém a minha solução para o Desafio de Programação de Lógica desenvolvido durante o bootcamp na DIO (Digital Innovation One).

## 📌 Objetivo do Projeto

O objetivo deste desafio é aplicar os conceitos essenciais da lógica de programação utilizando JavaScript, tais como:
* Variáveis e Operadores
* Estruturas Condicionais (`if`, `else if`, `else`)
* Laços de Repetição (`for`, `while`)
* Funções e Organização de Código

---

## 🛠️ Tecnologias Utilizadas

* **JavaScript (Node.js)**: Para a execução da lógica do projeto.
* **Visual Studio Code**: O editor de código utilizado no desenvolvimento.
* **Git & GitHub**: Para controle de versão e gerenciamento do repositório.
